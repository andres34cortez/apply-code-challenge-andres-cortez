# Prueba Técnica de Frontend



### Desarrollo Local

1. Instala las dependencias:
```bash
npm install
# o
pnpm install
```

2. Ejecuta el servidor de desarrollo:
```bash
npm run dev
# o
pnpm dev
```

3. [http://localhost:3000](http://localhost:3000) en  navegador.

### Variables de Entorno

La aplicación maneja automáticamente las URL de la API:
- **Desarrollo local**: Usa `http://localhost:3000` por defecto
- **Producción (Vercel)**: Usa automáticamente URLs relativas (mismo dominio) - https://apply-code-challenge-andres-cortez.vercel.app/
